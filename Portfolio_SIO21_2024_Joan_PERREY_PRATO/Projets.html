Projet Perso

Dans cette section, je vous propose de découvrir une sélection de mes projets personnels.

Jeu à Enigme

HTML | JS | CSS

Pour ce projet, j'ai voulu faire un jeu à 2 d'énigme à code.
Le but du jeu est de deviner le code de l'autre joueur.
En premier lieu, les 2 joueurs rentre leur code à 4 chiffres bien évidemment caché par le type "password".
Puis chacun leurs tour ils essayent de deviner le code de leur adversaire.
A chaque tour 2 indices sont donné sur le code essayé :
- si un ou plusieurs chiffres est bien placé,
- sinon si un ou plusieurs chiffres du code essayé sont dans le code à devnier.
Vous pourrez le retrouver sur mon Github.
